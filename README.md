**Catatanku** merupakan _Notes Apps_ yang berbasis website yang dibangun menggunakan React Js. Catatanku memiliki fitur utama yaitu
menambahakan, mengubah, menghapus data catatan yang user buat, serta fitur pencarian untuk memudahakan user mencari catatan yang sudah dibuat berdasarkan judul. Walaupun berbasis
web Catatanku dapat diinstall langsung pada browser, **_karena sudah mengadaptasi teknologi PWA
(Progressive Web Apps)_**

Anda dapat menggunakan atau berkontribusi pada aplikasi ini. Jika anda ingin menggunakan aplikasi Catatanku anda bisa mengklik tautan ini [**Catatanku | Notes Apps**](https://znovandap-portfolio.netlify.app/). Namun Jika anda ingin berkontribusi untuk mengembangkan aplikasi ini, anda dapat melakukan fork pada repository ini

```bash
npm install

npm run dev
